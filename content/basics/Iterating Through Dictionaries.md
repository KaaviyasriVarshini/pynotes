```python
user = {"name": "Alice", "age": 30, "city": "London"}

for key in user:
    print(key, ":", user[key])

for key, value in user.items():
    print(key, "=>", value)
```

    name : Alice
    age : 30
    city : London
    name => Alice
    age => 30
    city => London
