```python
text = "python"

# String to list of characters
char_list = list(text)
print(char_list)  # Output: ['p', 'y', 't', 'h', 'o', 'n']

# List of strings to single string
words = ["Python", "Type", "Conversion"]
joined = " ".join(words)
print(joined)     # Output: Python Type Conversion
```
