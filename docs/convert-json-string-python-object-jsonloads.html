Convert Json String → Python Object (Json.Loads)

Sun 25 January 2026 
import json

json_text = '{"name": "Bob", "age": 25}'
python_obj = json.loads(json_text)

print(python_obj)
print(type(python_obj))  # dict

{'name': 'Bob', 'age': 25}
<class 'dict'>
