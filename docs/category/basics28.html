Modifying Enclosing Variables With Nonlocal
Sun 25 January 2026 
def counter():
    value = 0
    def increment():
        nonlocal value
        value += 1
        return value
    return increment

count = counter()
print(count())
print(count())

1
2

    def adder(y):
